*Customer:*

- All included components are the responsibility of the customer.
- The customer provides the VM or server and must install and configure Docker and Docker Compose on the server.
- The customer provides a directory (SHARED_FOLDER) with sufficient read/write permissions for the Docker containers.
- The customer provides the LLM provider.
- The customer configures the proxy connection (TLS/HTTPS).
- The customer configures the necessary firewall rules.
- The customer configures integration with the existing authentication system.
- The customer handles backup and recovery (database, SHARED_FOLDER, configuration files).


*{application}*

We provide the application as a Docker image. This includes, and is installed via Docker Compose, 
the individual containers ({application} backend, LLM API, RAG API, Caddy proxy, Redis, Celery worker, customer-specific MCP server API).
