

=== Erforderliche Konfigurationen

==== Welche Login-Methode soll verwendet werden?

{application} unterstützt:

* eine OpenID Connect Authentifizierung mit Microsoft Entra ID
* eine SAML Authentifizierung mit Microsoft Entra ID
* eine lokale Benutzer-/Passwortverwaltung

==== Erreichbarkeit (HTTP/S, Reverse Proxy, SSL-Zertifikate)

*Erforderliche Ports:*

|===
| Port | Protokoll | Verwendung | Erforderlich für 

| **80** 	| HTTP 	| HTTP-zu-HTTPS-Redirect, Let's Encrypt ACME-Challenge 	| Produktion mit Caddy 
| **443** 	| HTTPS | Verschlüsselter Zugriff (TLS) 						| Produktion (empfohlen) 
| **8000** 	| HTTP 	| Direktzugriff auf Anwendung 							| Test/Entwicklung
|===

*Empfehlung:* Einsatz eines Reverse Proxys (z. B. Caddy, Nginx) mit TLS-Zertifikaten.

{application} unterstützt:

* Caddy mit Let's Encrypt (Reverse Proxy)
* DNS‑01 (öffentlich trusted, ohne Freigabe 80/443)
* ohne öffentlichen DNS (keine Portfreigabe)
