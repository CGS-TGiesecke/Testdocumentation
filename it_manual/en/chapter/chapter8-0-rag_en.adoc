
=== Database and Storage

Data storage:

* SQLite: User data, metadata, logs
* SHARED_FOLDER: Uploaded documents, configuration files, and mount for Docker
* ChromaDB: Embeddings for RAG

*All data resides on the customer’s server.*

RAG (Retrieval-Augmented Generation) enables the use of current, verifiable information in
ifeval::[{cgs-assist} == 1]
 KI Chat
endif::[]
ifeval::[{arc-assist} == 1]
 AI Chat
endif::[]
 using your own data basis and provides transparency in the document view regarding the source of the information.
The application {application} supports various options for data integration, which can be configured according to 
the requirements present at the customer's site in preparation for delivery.

{application} supports:

* Shared Folder (SHARED_FOLDER)
* SharePoint connection with Microsoft Entra ID
