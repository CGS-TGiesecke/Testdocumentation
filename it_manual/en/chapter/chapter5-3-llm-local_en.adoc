
==== Local Models (for Highest Data Protection Requirements)

There are also options available for customers who do not want cloud connectivity.
The platform {application} supports local LLMs via *OpenAI-compatible APIs*.

===== Supported tools

|===
| Tool | Note

| Ollama
| the simplest solution

| vLLM
| highest performance

| LM Studio
| GUI-based
|===
