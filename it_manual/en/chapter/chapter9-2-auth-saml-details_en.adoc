===== Registering the application in the Microsoft Entra Admin Center for SAML

*Login*

Sign in to the Microsoft Entra Admin Center using an account that has administrator permissions for your organization.
https://entra.microsoft.com

*Navigation to "App registrations"*

* Select Entra ID on the far left
* Navigate to App registrations
* Register a new application
* Click New registration (top menu)

*Enter application information*

* Name: Choose a name for the application. The name will be visible later. Suggestion: {application}-client-app
Supported account types: Decide whether the application should support internal users only (your own organization), external users, or multiple organizations. Default is “Accounts in this organizational directory only”.
* Redirect‑URI: Must be identical to the value "redirect_uri" in the init_config.yaml.

*Create*

* Click Register.

*Enable SAML as the authentication method*

* In the application, on the left-hand side, go to Single sign-on
* Select SAML
* The SAML‑based configuration page will open

*Basic SAML Configuration*

* Identifier (Entity ID): The unique identifier in the Admin Center
* Redirect URL: Must match the value of "redirect_uri" in the init_config.yaml

*Attributes & Claims*
These values must be configured so that user information is passed to the {application} during each login.
This ensures that the application always contains the latest user data (first name, last name, email).

*SAML Certificates*

The provided “Federation Metadata XML” must be downloaded and placed in the SHARED_FOLDER, 
and must match the value "idp_metadata_path" in the init_config.yaml.

The file contains:

* Entity ID
* SSO URL
* Certificates for validating signatures

*Configuration data:*

** Application ID: *Client ID*
** Tenant ID: *Tenant ID*
** Client secret: *Client Secret*, can be time‑limited
** Redirect‑URI: *Redirect‑URI* redirect after successful login to {application}
```
Note: Be sure to record/copy these values for your application configuration in `init_config.yaml`.
```
