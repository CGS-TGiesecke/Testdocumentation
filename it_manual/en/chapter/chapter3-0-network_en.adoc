
==== Network / Ports

* *For cloud LLM providers (Azure OpenAI, AWS Bedrock):* Permanent internet connection required
* *For local models:* Internet connection only required during installation (Docker image download)

*Proxy support:* The solution supports operation behind a proxy. Proxy configuration can be set via environment variables.

*Required Ports:*

|===
| Port   | Protocol | Purpose                                   | Required for

| **80**   | HTTP     | HTTP-to-HTTPS redirect, Let's Encrypt ACME challenge | Production with Caddy
| **443**  | HTTPS    | Encrypted access (TLS)                   | Production (recommended)
|===

*Production recommendation:* Use a reverse proxy (e.g., Caddy, Nginx) with TLS certificates.
