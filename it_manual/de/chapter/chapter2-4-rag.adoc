
=== RAG (Retrieval-Augmented Generation)

Das RAG bietet die Möglichkeit zur Erweiterung der Wissensuche im
ifeval::[{cgs-assist} == 1]
 KI Chat
endif::[]
ifeval::[{arc-assist} == 1]
 AI Chat
endif::[]
 oder der Nutzung von Use Cases zusätzliche Verzeichnisse aus dem Kundenintranet mit einzubeziehen.
Die Applikation {application} unterstützt verschiedene Möglichkeiten der Datenanbiendung, die in Vorbereitung einer Auslieferung,
nach den beim Kunden vorhandenen Anforderungen, konfiguriert werden können.
 
{application} unterstützt:

* integriertes DMS
* Sharepoint Anbindung über die Microsoft Entra ID
