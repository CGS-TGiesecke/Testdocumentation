
===  Datenbank und Storage

Datenhaltung:

* SQLite: Benutzerdaten, Metadaten, Logs
* SHARED_FOLDER: Hochgeladene Dokumente, Konfigurationsdateien und Mount zum Docker
* ChromaDB: Embeddings für RAG

*Alle Daten liegen auf dem Kundenserver.*

Das RAG (Retrieval-Augmented Generation) bietet die Möglichkeit zur Erweiterung der Informationsquellen im
ifeval::[{cgs-assist} == 1]
 KI Chat
endif::[]
ifeval::[{arc-assist} == 1]
 AI Chat
endif::[]
 oder bei der Nutzung von Use Cases um zusätzliche Verzeichnisse aus dem Kundenintranet.
Die Applikation {application} unterstützt verschiedene Möglichkeiten der Datenanbindung, die in Vorbereitung einer Auslieferung,
nach den beim Kunden vorhandenen Anforderungen, konfiguriert werden können.
 
{application} unterstützt:

* Shared Folder (SHARED_FOLDER)
* Sharepoint Anbindung über die Microsoft Entra ID
