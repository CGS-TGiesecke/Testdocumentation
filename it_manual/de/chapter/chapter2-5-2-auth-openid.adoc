
==== OpenID Authentifizierung mit Microsoft Entra ID

Die OpenID-Authentifizierung mit Microsoft Entra ID wird heute für alle professionellen, cloud- und unternehmensorientierten Anwendungen empfohlen, 
um Sicherheit, Nutzerkomfort und Compliance-Anforderungen adäquat adressieren zu können. Sie bringt folgende Vorteile und ist in vielen Szenarien notwendig:

*Einheitlicher Anmeldeprozess (Single Sign-On, SSO)*

* Benutzer können sich mit ihren Unternehmensanmeldedaten (z. B. E-Mail/Passwort bei Microsoft 365) in verschiedensten Anwendungen anmelden.
* Mehrere Apps (Cloud und On-Premises) nutzen ein zentrales Identitätsmanagement.
* Passwort-/Benutzerverwaltung erfolgt einheitlich.

*Sicherheitsgewinn & Compliance*

* Microsoft Entra ID unterstützt Multi-Faktor-Authentifizierung (MFA), Conditional Access und Identity Protection.
* Sicherheits- und Richtlinienkontrolle bis auf Anwendungsebene.
* Auditierbarkeit, Protokollierung und zentrale Policy-Verwaltung.

*Standardisierung & Integration*

* OpenID Connect ist ein weit verbreiteter offener Authentifizierungsstandard – kompatibel mit modernen Frameworks und Bibliotheken.
* Integration von Drittanbieter-Anwendungen und Eigenentwicklungen möglich.
* Minimierung von Integrations- und Wartungsaufwand.

*Notwendigkeit*

* **Datenschutz und Compliance:** Viele Branchen/Unternehmen fordern zentrale Identitätsdienste und revisionssichere Authentifizierung.
* **Reduktion von Schatten-IT:** Eine einheitliche Authentifizierung erschwert un-autorisierte Zugangslösungen.
* **Anwenderkomfort:** Weniger Anmeldeprobleme, Self-Service bei Passwortänderungen, SSO über viele Systeme hinweg.

===== Voraussetzungen

* Ein Azure-Mandant und Zugriff auf das Microsoft Entra Admin Center: https://entra.microsoft.com
* Eine registrierte Anwendung in Microsoft Entra ID (siehe "Registrierung einer Anwendung im Microsoft Entra ID Admin Center")
* Folgende Informationen/Endpunkte:
  - Tenant ID
  - Client ID (Anwendungs-ID)
  - Client Secret (falls für den Flow benötigt)
  - Redirect URI (muss in der App-Registrierung hinterlegt werden)
