
==== Ordnerverwaltung

Diese Seite zeigt alle Ordner an. Die Berechtigungen für Ordner werden über Rollen zugeordnet, wodurch die Sichtbarkeit im Bereich „Dokumente“ gesteuert wird.
Einzelne Ordner können, nach Bestätigung einer Sicherheitsabfrage, gelöscht werden. Im Ordner befindliche Dokumente werden mit gelöscht.

image::../images/Abbildung-20.jpg[Administration - Ordnerverwaltung, title="Administration - Ordnerverwaltung", width=600]

Im Dialog zur Sicherheitsabfrage werden die enthaltenen und mit gelöschten Dokumente aufgelistet.

image::../images/Abbildung-51.jpg[Administration - Ordnerverwaltung - Sicherheitsabfrage, title="Administration - Ordnerverwaltung - Sicherheitsabfrage", width=600]

*Erlaubte Dokumente* sind: .pdf .docx .xlsx, .pptx, .txt.
