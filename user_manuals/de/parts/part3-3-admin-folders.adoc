
==== Ordnerverwaltung

Auf dieser Seite werden alle angelegtenkönnen Ordner angezteigt. Diesen Ordnern können hier Berechtigungen durch die Rollen zugewiesen werden. 
Dadurch wird die Sichtbarkeit und der Zugriff auf diese Ordner unter Navigationsbereich „Dokumente“ gesteuert.

Vom System erlaubte Dokumente zur Nutzung in der Ordnerverwaltung sind: .pdf .docx .xlsx, .pptx, .txt.
