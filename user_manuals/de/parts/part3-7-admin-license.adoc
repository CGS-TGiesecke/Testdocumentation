
==== Lizenzverwaltung

Hier kann die Lizenz verwaltet werden. Es besteht die Möglichkeit der Aktualisierung, wenn die vorhandene ausgetauscht wurde.
Auch der Kontakt per Mail zum Assist Service ist von hier aus möglich
