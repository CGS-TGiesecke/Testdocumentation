
==== Lizenzverwaltung

Auf dieser Seite kann die für den Betrieb der Applikation {application} notwendige Lizenz verwaltet werden. 
Es besteht die Möglichkeit der Aktualisierung, wenn eine vorhandene Lizenz ausgetauscht wurde.
Auch der Kontakt per Mail zum {application} Support ist auf dieser Seite integriert. Beim Klick auf den Button öffnet der 
Standard Mail Client des Benutzers mit vor eingestelltem Empfänger.

image::../images/Abbildung-30.jpg[Administration - Lizenzverwaltung, title="Administration - Lizenzverwaltung", width=500]

Es kann nach Auswahl eine gänzlich neue Lizenz hochgeladen werden.

image::../images/Abbildung-31.jpg[Administration - Lizenzverwaltung - Lizenz laden, title="Administration - Lizenzverwaltung - Lizenz laden", width=300]
