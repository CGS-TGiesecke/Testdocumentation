
==== Portal-Seite

Auf dieser Seite können verschiedene Verknüpfungen verwaltet werden. Es besteht die Möglichkeit zur Neuanlage, zum Löschen und 
Bearbeiten einzelner Einträge. Die Verfügbarkeit einzelner Verlinkungen kann über seperate Berechtigungen gesteuert werden.
Zusätzlich kann die gesamte Portalseite aktiviert/deaktiviert werden und diese ist dann auch nicht mehr in der Navigation verfügbar.

image::../images/Abbildung-41.jpg[Administration - Portal-Seite, title="Administration - Portal-Seite", width=500]

Es können Seiten oder Applikationen verknüpft werden. Die Verknüpfung von Dokumenten auf Laufwerden oder Shares ist nicht möglich.
