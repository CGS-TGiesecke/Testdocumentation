
==== Role Management

Roles can be created and edited on this page. When creating a new role, a system‑wide unique role name must be assigned. Each role may also include a description. Both the name and description can be changed at any time.

image::../images/Abbildung-18.jpg[Administration - Role Management, title="Administration - Role Management", width=400]

Roles cannot be deactivated and can only be removed from the overview after confirming a security prompt. System‑defined roles cannot be deleted.

image::../images/Abbildung-19.jpg[Administration - Role Management - edit role, title="Administration - Role Management - edit role", width=400]
