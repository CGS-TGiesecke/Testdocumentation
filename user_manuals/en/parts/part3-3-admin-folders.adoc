
==== Folder Management

On this page, all created folders can be displayed. Permissions can be assigned to these folders through roles here. 
This controls the visibility and access to these folders within the "Documents" navigation area.

The document formats permitted by the system for use in the folder management are: .pdf, .docx, .xlsx, .pptx, and .txt.
