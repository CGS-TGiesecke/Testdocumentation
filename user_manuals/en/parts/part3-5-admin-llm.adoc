
==== LLMs and Embedding Models

All LLM providers stored in the system are listed and managed here. These providers supply access to Large Language Models (LLMs) and maintain the infrastructure required to operate advanced AI language models.

image::../images/Abbildung-22.jpg[Administration - LLMs and Embedding Models, title="Administration - LLMs and Embedding Models", width=500]

New providers and models can be created or, after a security prompt, removed again. You can also preselect whether an AWS, Azure, or local provider should be created.

image::../images/Abbildung-32.jpg[Administration - LLMs and Embedding Models - Create Provider, title="Administration - LLMs and Embedding Models - Create Provider", width=250]

By clicking the arrow on the left, the provider configuration can be opened, and required models may be created or removed (removal also requires a confirmed security prompt).
